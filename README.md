# Ejercicios de Python - 2º DAW

Este repositorio forma parte del trabajo de **Digitalización**.  
El objetivo es recopilar y subir todos los ejercicios realizados en la asignatura de **Programación en Python** al repositorio de GitHub, para facilitar el acceso, la organización y el versionado del código.

## Estructura del repositorio
- `Python/` → Carpeta principal con los ejercicios.
- Subcarpetas organizadas por temas (funciones, módulos, expresiones regulares, etc.).
- Archivos individuales como `recursividad.py`, `modulo.py`, etc.

## Objetivos
- Digitalizar y centralizar los ejercicios de Python.
- Practicar el uso de **Git** y **GitHub** como herramientas de control de versiones.
- Facilitar la colaboración y el seguimiento de los avances.
